# Information
Robocop Data East arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jtcop

# How to run
Needs the robocop.rom and a proper ARC file in the core or SD root folder
You need to either update rename the core file to jtcop.rbf or change its reference in the ARC file accordingly
