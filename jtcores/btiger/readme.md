# Information
Black Tiger arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jttora

# How to run
Needs the blktiger.rom and a proper ARC file in the core or SD root folder
You need to either update/rename the core file to jtbtiger.rbf or change its reference in the ARC file accordingly
