# Information
Ghost'n Goblins arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jtgng

# How to run
Needs the gng.rom and a proper ARC file in the core or SD root folder
You need to either update/rename the core file to jtgng.rbf or change its reference in the ARC file accordingly
