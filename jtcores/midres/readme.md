# Information
Midnight Resistance arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jtmidres

# How to run
Needs the midres.rom and a proper ARC file in the core or SD root folder
You need to either update/rename the core file to jtmidres.rbf or change its reference in the ARC file accordingly
