# Information
Double Dragon arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jtdd

# How to run
Needs the dd.rom and a proper ARC file in the core or SD root folder
You need to either update/rename the core file to jtdd.rbf or change its reference in the ARC file accordingly
