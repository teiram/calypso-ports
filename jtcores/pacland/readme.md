# Information
Pacland arcade core ported from [jotego](https://github.com/jotego/jtcores.git)

Core Id: jtpaclan

# How to run
Needs the pacland.rom and a proper ARC file in the core or SD root folder
You need to either update/rename the core file to jtpaclan.rbf or change its reference in the ARC file accordingly
