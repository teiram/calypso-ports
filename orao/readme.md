# Information
Orao, old Croatian computer from the 80s ported from [MiSTer](https://github.com/MiSTer-devel/Orao_MiSTer)

# How to run
No ROM files are needed. 
- To go into BASIC from monitor, type: BC
- To load tapes, first select it in the OSD, then from basic: LMEM ""

# Known issues
Seems that PACMAN doesn't load and the keyboard doesn't properly handle pressing a key before releasing the previous one. Don't know how the original hardware worked tho
